# Employee Attrition Analysis

## Project Overview

This project analyzes employee attrition and predicts whether an employee is likely to leave the company using Machine Learning techniques. The goal is to help HR departments identify employees at risk of leaving and improve employee retention strategies.

## Dataset

IBM HR Analytics Employee Attrition Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost
* Jupyter Notebook

## Exploratory Data Analysis (EDA)

The following analyses were performed:

* Employee Attrition Distribution
* Age vs Attrition
* Monthly Income vs Attrition
* Department vs Attrition
* Job Role vs Attrition
* Gender vs Attrition
* Overtime vs Attrition

## Machine Learning Models

The following classification models were used:

1. Logistic Regression
2. Random Forest Classifier
3. XGBoost Classifier

## Evaluation Metrics

The models were evaluated using:

* Accuracy
* Recall
* ROC-AUC Score
* Classification Report
* Confusion Matrix

## Dashboard KPIs

* Total Employees
* Employees Left
* Attrition Rate
* Average Monthly Income

## Key Findings

* Employees working overtime showed higher attrition.
* Monthly income influenced employee retention.
* Certain departments experienced higher employee turnover.
* Age and job role affected attrition behavior.
* Machine Learning models successfully predicted employee attrition.
